# Sales Performance Dashboard

## Project Overview

This project is an interactive Power BI dashboard developed to analyze sales performance and generate business insights. It provides a comprehensive view of key business metrics through dynamic visualizations and interactive filters, enabling data-driven decision making.

---

## Objectives

- Analyze sales performance across different dimensions.
- Monitor key business KPIs.
- Identify sales and profit trends.
- Evaluate category and sub-category performance.
- Support business decisions using interactive dashboards.

---

## Dataset

The dashboard is built using two CSV datasets:

- Orders.csv
- Details.csv

The datasets are related using **Order ID**.

---

## Dashboard Features

- KPI Cards
  - Total Sales
  - Total Profit
  - Total Quantity
  - Average Order Value

- Monthly Profit Analysis
- State-wise Sales Analysis
- Category & Sub-Category Performance
- Payment Mode Distribution
- Quarterly Analysis
- Interactive Filters (Quarter & State)

---

## Tools & Technologies

- Power BI
- Power Query
- DAX
- CSV Dataset

---

## Business Insights

- Identified top-performing product categories.
- Compared monthly profit trends.
- Analyzed customer purchasing behavior by payment mode.
- Evaluated regional sales performance.
- Tracked key business KPIs through interactive visualizations.
